# IP Extractor

[🇺🇸 English Version](README.md)

ابزار استخراج آی‌پی از فایل‌های اسکنر، لاگ و فایل‌های متنی.

## امکانات

- استخراج IPv4 از فایل‌های TXT، LOG و CSV
- حذف پورت از آی‌پی‌ها
- حذف آی‌پی‌های تکراری
- فیلتر کردن فقط موارد Healthy
- محدود کردن تعداد خروجی
- کپی مستقیم خروجی
- دانلود فایل خروجی
- بدون نیاز به بک‌اند
- پشتیبانی از Drag & Drop

## نحوه استفاده

1. فایل را انتخاب یا Drag & Drop کنید.
2. گزینه‌های مورد نظر را فعال کنید.
3. روی دکمه استخراج کلیک کنید.
4. خروجی را کپی یا دانلود کنید.
